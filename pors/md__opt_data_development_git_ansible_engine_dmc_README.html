Ansible Engine ?

Up to v3.9 the whole project was named "Ansible Engine (AE)" which unfortunately conflicted with RedHat's product name: ansible engine. That's why the whole project has been renamed in v4 to "Playbooks-On-Rails for Splunk (PORS)".
