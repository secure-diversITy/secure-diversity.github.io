About / README

PORS is a collection of Ansible playbooks, roles & tasks for installing, configuring & managing your whole Splunk infrastructure - including Apps using the Splunk deployment process. But the icing on the cake is that PORS comes with a terminal UI that does all this with the snap of a finger!

Ansible Engine ?

Up to v3.9 the whole project was named "Ansible Engine (AE)" which unfortunately conflicted with RedHat's product name: ansible engine. That's why the whole project has been renamed in v4 to "Playbooks-On-Rails for Splunk (PORS)".
